Mathematical Calculator

This is a mathematical calculator that can help you solve some tough math equations. Some of these equations include:

Simplify
Ex: 2^2 + 2(2) = 8
Factor
Ex: x^2 - 5x = x (x - 5)
Derive
Ex: 4x^3 = 12x^2
Integrate
Ex: x^2 = 1/3x^3 + C
Find 0's
Ex: x^2 - 5x + 4 = [4, 1]
Trig Functions
Ex: sin(pi) = 0
